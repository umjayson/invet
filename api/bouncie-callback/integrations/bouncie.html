How to Set Up

Follow these steps to connect your Bouncie account with InvetChat:

1

Enter your API Key

Click "Edit API Key" above and enter your Bouncie API key.

2

Connect your Account

Click the "Connect to Bouncie" button to authorize InvetChat to access your Bouncie data.

3

Test the Connection

Use the "Test Connection" button to verify that everything is working correctly.

4

Start Tracking

Once connected, you'll be able to view and manage your vehicles in the InvetChat dashboard.
